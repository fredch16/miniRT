To pull files ↙️
```bash
git clone git@github.com:fredch16/miniRT.git
```
To compile the program ↙️

```bash
$ make
```
To load you have to launch the program with a valid .rt scene file as a parameter 🗺️
```bash
$ ./miniRT ./scenes/scene.rt
```
